# Arrays Problems

This folder contains LeetCode problems related to Arrays.

## ✅ Problems Solved

1. Two Sum
   - Pattern: HashMap
   - Time Complexity: O(n)
   - Space Complexity: O(n)

2. Contains Duplicate
   - Pattern: HashSet
   - Time Complexity: O(n)
   - Space Complexity: O(n)

3. Anagram
   - Pattern: Array
   - Time Complexity: O(n)
   - Space Complexity: O(1)

3. Valid Palindrome
   - Pattern: Two Pointer
   - Time Complexity: O(n)
   - Space Complexity: O(1)
